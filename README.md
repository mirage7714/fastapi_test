# Simple FastAPI sample
 
* Installation  
`pip install -r requirements.txt`  

* Run the application  
`uvicorn main:app --reload`  

* API Endpoints  
- Get all users  
`GET /users/`  

- Get user via id  
`GET /users/{user_id}  

- Add a new user  
`POST /users/`

- Delete a user  
`DELETE /users/{user_id}
